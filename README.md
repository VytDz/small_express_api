# small_express_api
**This is a small express and postgresql CRUD api**

Requires to have node installed on your machine.
Requires postgreSql installed on your machine.
Requires a config.json file, which hold your credentials:

{
    "user": "user",
    "host": "localhost",
    "database": "api",
    "password": "password",
    "port": 5432
}
